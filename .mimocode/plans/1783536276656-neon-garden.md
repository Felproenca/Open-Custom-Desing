# Plano: Deploy Automático via FTP para HostGator

## Objetivo
Criar script de deploy que envia automaticamente os arquivos do site para o HostGator via FTP/FTPS.

- **Diretório remoto:** public_html/

## Arquivos a enviar
```
public_html/
├── index.html
├── sitemap.xml
├── robots.txt
├── api/
│   └── market.php
├── assets/
│   └── images/ (14 imagens)
├── blog/ (5 páginas HTML)
└── blog.css
```

## Implementação

### 1. Script PowerShell de deploy
- Arquivo: `references/fortunato-trader/deploy.ps1`
- Usa `Invoke-WebRequest` ou `curl` com FTPS
- Faz upload incremental (só arquivos modificados)
- Log de resultado

### 2. Comando de deploy
```powershell
# Deploy manual
.\deploy.ps1

# Deploy via pnpm script
pnpm deploy-hostgator
```

### 3. Fluxo de trabalho
1. Fazer alterações no código local
2. Rodar `.\deploy.ps1` ou usar atalho
3. Script conecta ao FTP e envia arquivos
4. Site atualizado automaticamente
