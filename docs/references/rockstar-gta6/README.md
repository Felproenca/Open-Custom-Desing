## Padrões visuais (transferíveis)

| Padrão | Descrição | Transferível? |
|--------|-----------|---------------|
| **Hero fullscreen** | Background full com conteúdo sobreposto | Sim |
| **Tipografia massiva** | Títulos gigantes com weight variado | Sim |
| **Seções alternadas** | Fundo claro/escuro alternando | Sim |
| **Grid de features** | Cards com imagem + texto | Sim |
| **Video embed** | Seções com vídeo de background | Sim |
| **Countdown/timer** | Contador regressivo para lançamento | Sim |
| **Parallax scroll** | Efeito de profundidade no scroll | Sim |
| **GSAP animations** | Animações complexas de scroll | Parcial (precisa lib) |
| **WebGL** | Backgrounds 3D interativos | Não (precisa Three.js) |

## Cores do sistema

- Fundo escuro: preto/dark (#000)
- Cor principal: laranja GTA (#FF6900 approximate)
- Texto: branco

## O que NÃO transferir

- Conteúdo proprietário (GTA, Rockstar)
- WebGL/Three.js scenes
- GSAP ScrollTrigger complexo (dependência pesada)
- Assets de jogo (imagens, vídeos)
